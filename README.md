## 🚀 Quick Start // البداية السريعة

1. **Clone the Repository // استنساخ المصدر:**
   ```bash
   git clone https://github.com/Moeabdelaziz007/Gemigram.git
   cd Gemigram
   ```

2. **Supply Global Credentials // توفير الاعتمادات العالمية:**
   Create a `.env.local` file with your Firebase and Gemini keys. Use `.env.example` as a template.
   قم بإنشاء ملف `.env.local` يحتوي على مفاتيح Firebase و Gemini الخاصة بك. استعن بملف `.env.example`.

3. **Ignition Sequence // تسلسل الإشعال:**
   ```bash
   npm install
   npm run dev
   ```

4. **Access the System // الوصول للنظام:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
